# Arise WhatsApp Bot

## Deploy on Render

1. GitHub pe fork karo ya new repo banao, ye files push karo
2. Render.com pe jao → New Web Service
3. Repo connect karo
4. Settings:
   - Build Command: `npm install`
   - Start Command: `node index.js`
   - Plan: Free (ya Starter $7/month for better uptime)

## Connect with Arise App

1. Deploy ke baad server URL copy karo (e.g. `https://your-bot.onrender.com`)
2. Arise app mein WhatsApp Bot section mein URL paste karo
3. Connect karo — Arise automatically register ho jaayega
4. Apna WhatsApp number daal ke pairing code generate karo
5. WhatsApp > Linked Devices > Link with phone number > Code enter karo

## That's it! Arise se contacts add/remove/toggle karo.
